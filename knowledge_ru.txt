sin 7
Определение: sin(x) - это отношение противолежащего катета к гипотенузе в прямоугольном треугольнике, т.е. противолежащий/гипотенуза
Разложение: sin(x)= x - x^3/3! + x^5/5! - ... + (-1)^n*x^(2n+1)/(2n+1)!
Свойства: sin^2(x) + cos^2(x) = 1, sin(-x) = -sin(x), sin(x + pi) = -sin(x)
Область_определения: При действительных x, область значений [-1, 1]
Ключевые_значения: sin(0) = 0, sin(pi/6) = 0.5, sin(pi/4) = sqrt(2)/2, sin(pi/3) = sqrt(3)/2, sin(pi/2) = 1
Предварительные_знания: Для изучения разложения - ряд Тейлора, для свойств - теорема Пифагора и самостоятельный вывод.
Примеры_использования: sin(pi/6), sin(x)^2+cos(x)^2 (необходимо определить x или подставить значение)
cos 7
Определение: cos(x) - это отношение прилежащего катета к гипотенузе в прямоугольном треугольнике, т.е. прилежащий/гипотенуза
Разложение: 1 - x^2/2! + x^4/4! - ... + (-1)^n*x^(2n)/(2n)!
Свойства: cos^2(x) + sin^2(x) = 1, cos(-x) = cos(x), cos(x + pi) = -cos(x)
Область_определения: При действительных x, область значений [-1, 1]
Ключевые_значения: cos(0) = 1, cos(pi/6) = sqrt(3)/2, cos(pi/4) = sqrt(2)/2, cos(pi/2) = 0
Предварительные_знания: Для изучения разложения - ряд Тейлора, для свойств - теорема Пифагора и самостоятельный вывод.
Примеры_использования: cos(pi/6), cos(x+pi) (необходимо определить x или подставить значение)
tan 7
Определение: tan(x) = sin(x)/cos(x) = противолежащий/прилежащий
Разложение: x + x^3/3 + 2x^5/15 + ... (|x| < pi/2)
Свойства: 1 + tan^2(x) = sec^2(x) = 1/cos^2(x), tan(-x) = -tan(x), tan(x + pi) = tan(x)
Область_определения: Все действительные числа, кроме pi/2 + k*pi (k - целое)
Ключевые_значения: tan(0) = 0, tan(pi/6) = 1/sqrt(3), tan(pi/4) = 1, tan(pi/3) = sqrt(3), tan(pi/2)->\\infty
Предварительные_знания: Для изучения разложения - ряд Тейлора, для свойств - теорема Пифагора и самостоятельный вывод.
Примеры_использования: tan(pi/6), 1+tan(x)^2 (необходимо определить x или подставить значение)
arcsin 7
Определение: Обратная функция к sin(x) (т.е. sin^-1)
Область_значений: [-pi/2, pi/2]
Свойства: arcsin(-x) = -arcsin(x), sin(arcsin(x)) = x (x in [-1,1])
Область_определения: [-1, 1]
Ключевые_значения: arcsin(0) = 0, arcsin(0.5) = pi/6, arcsin(1) = pi/2
Предварительные_знания: Изучить sin(x)
Примеры_использования: arcsin(1), sin(arcsin(sin(arcsin(x)))) (необходимо определить x или подставить значение)
arccos 7
Определение: Обратная функция к cos(x) (т.е. cos^-1)
Область_значений: [0, pi]
Свойства: arccos(-x) = pi - arccos(x), cos(arccos(x)) = x (x in [-1,1])
Область_определения: [-1, 1]
Ключевые_значения: arccos(0) = pi/2, arccos(0.5) = pi/3, arccos(1) = 0
Предварительные_знания: cos(x), самостоятельный вывод свойств.
Примеры_использования: arccos(-0.5), arccos(x)+arcsin(x) (необходимо определить x или подставить значение)
arctan 7
Определение: Обратная функция к tan(x) (т.е. tan^-1)
Область_значений: (-pi/2, pi/2)
Свойства: arctan(-x) = -arctan(x), tan(arctan(x)) = x (x in R)
Область_определения: R
Ключевые_значения: arctan(0) = 0, arctan(1/sqrt(3)) = pi/6, arctan(1) = pi/4
Предварительные_знания: tan(x)
Примеры_использования: arctan(2)
sinh 7
Определение: Гиперболический синус: (e^x - e^-x)/2
Разложение: x + x^3/3! + x^5/5! + ...
Свойства: cosh^2(x) - sinh^2(x) = 1, sinh(-x) = -sinh(x)
Область_определения: R, область_значений: R
Ключевые_значения: sinh(0) = 0, экспоненциальный рост при увеличении x
Предварительные_знания: Экспоненциальная функция
Примеры_использования: cosh(x)^2-sinh(x)^2 (необходимо определить x или подставить значение)
cosh 7
Определение: Гиперболический косинус: (e^x + e^-x)/2
Разложение: 1 + x^2/2! + x^4/4! + ...
Свойства: cosh^2(x) - sinh^2(x) = 1, cosh(-x) = cosh(x)
Область_определения: R, область_значений: [1, +бесконечность]
Ключевые_значения: cosh(0) = 1, экспоненциальный рост при увеличении x
Предварительные_знания: Экспоненциальная функция
Примеры_использования: cosh(x)^2-sinh(x)^2 (необходимо определить x или подставить значение)
tanh 6
Определение: Гиперболический тангенс: sinh(x)/cosh(x)
Разложение: x - x^3/3 + 2x^5/15 - ... (|x| < pi/2)
Свойства: 1 - tanh^2(x) = sech^2(x), tanh(-x) = -tanh(x)
Область_определения: R, область_значений: (-1, 1)
Ключевые_значения: tanh(0) = 0, стремится к \pm 1 при x -> \pm бесконечность
Примеры_использования: 1-tanh(x)^2 (необходимо определить x или подставить значение)

Предварительные_знания: Линейные/квадратные уравнения, теорема Виета.

Матрицы 6
Определение: Прямоугольная таблица чисел размером m*n, обозначается заглавными буквами, напр. A = [a_11, a_12; a_21, a_22]

Производная 6
Определение: f'(x) = lim(h->0) [f(x+h)-f(x)]/h
Правила: Степень (d/dx x^n= nx^(n-1)), Произведение ((uv)'=u'v+uv'), Цепное (f(g(x))'=df/dg * dg/dx=f(g)'*g(x)')
Примеры: sin(x)->cos(x), e^x->e^x, ln(x)→1/x
Применение: Касательные, оптимизация (это обязательно надо учить!!!!)
Предварительные_знания: Для изучения пределов - пределы, в целом - функции.
